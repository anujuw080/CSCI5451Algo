What is the Knapsack Problem?




  • Given a set of items, where each item has its own mass and value, determine the number of each items to include in the collection that will return the largest value while having a total weight that is less than or equal to the limit.

Pseudocode:

 

KnapSack(v, w, n, W) {
  for (w = 0 to W) V[0, w] = 0;
  for (i = 1 to n)
  for(w = 0 to W)
   if (w[i] <= w)
    V[i, w] = max{V[i - 1, w], v[i] + V[i - 1, w - w[i]]};
   else
    V[i, w] = V[i - 1, w];
 return V[n, W];
}

 

  • Runtime = O(n log n) time
